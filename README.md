### Lee Davis

I build systems that make households run — AI agents, IoT hardware, and home automation.

#### Projects

**[household-agent](https://github.com/leedavis-irl/household-agent)** — A multi-channel AI agent that acts as a household chief of staff. Coordinates calendars, controls smart home devices via Home Assistant, manages finances through Monarch Money, and sends messages across Signal, Slack, and SMS. Built on Claude (Anthropic) with a four-flow architecture (message broker, brain, action router, response router), identity-first context loading, per-user permissions, and 40+ tool capabilities.
Node.js · Anthropic Claude · Home Assistant · EC2 · GitHub Actions CI/CD

**[task-tracker](https://github.com/leedavis-irl/task-tracker)** — A physical chore tracking system. ESP32 buttons (Seeed XIAO ESP32-C6) placed at task locations record completions via HTTP to a Flask server. A Waveshare 7.5" e-ink display polls the server and shows a live checklist. Daily summaries sent via Signal. Custom 3D-printed enclosures designed in OpenSCAD.
Python/Flask · ESPHome · ESP32 · OpenSCAD · EC2
